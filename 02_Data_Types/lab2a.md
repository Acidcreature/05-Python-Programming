## Lab 2A

Using the Python interpreter: find the type of the following variables. Feel free to experiment with other variables and such.

**Type of:​**

* 10​  -- Int
* 10.5​  -- Float
* "10"​  -- Str
* "Hello!"​  -- Str
* ""​​  -- Str
* ''​  -- Str
* True​ -- Bool
* 0​  -- Int
* type​  -- Type
* object​  --type
* b"10101101" **Try in Py2 and Py3**  -- P3, Bytes
* 0b10101101​ -- Int
* \[1,2,3\]​  -- list
* \(1,2,3\)​  -- tuple
* {1,2,3}​  -- Set
* {'one':1}​  -- Dict
* 5j​  -- Complex
