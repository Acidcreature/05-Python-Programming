# Numbers

### Prefixes

Prefixes convert types like **binary, hex and octal **into int. 

* No prefix for decimal​

* Prefix with 0b, B for binary \(ex. 0b10 == 2\)​

* Prefix with 0x, X for hex \(ex. 0xF == 15\)​

* Prefix with 0o, O for octal \(ex. 0o100 == 64\)​
